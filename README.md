# Raspberry
Projetos em Python para **raspberry Pi!**


## Funcionalidades:

Leitura de sensor de temperatura/umidade/pressão BME-280;
Calculo de ponto de orvalho.

Inserção em Banco de dados MySql das principais váriaveis do sensor.
